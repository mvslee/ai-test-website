**AI社交平台营销助手**

**产品定位**

+:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **核心场景：**                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                   |
| 场景一：大品牌pop-up，线下活动，行业展会等。主要的目的是为了帮助企业获得社媒KOC曝光量，提升KOC内容质量，提升品牌社媒搜索。这类型的企业所发布的内容均需要审核，为了避免不qualify的内容出街，同时也避免危机公关，大品牌比较爱惜羽毛，担心品牌危机。 |
|                                                                                                                                                                                                                                                   |
| 场景二：非现阶段priority。主要针对本地生活的商家，用于帮助商家提升社媒的曝光量。内容审核相对来说不严格。场景二对我们来说主要是走量的                                                                                                              |
+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

**核心功能**

快速配置内容，一键分发，让品牌营销内容高效触达全网：

内容快速配置：活动运营方可快速配置活动及素材，生成营销码，内容快速推送到用户侧（无aigc的能力）

用户极简发布：用户扫描二维码获得内容，可一键同步至小红书、微博、大众点评、朋友圈等社交平台

数据效果追踪：活动运营方可监控内容数据表现，优化营销配置

**目标用户**

品牌市场团队：需要高效管理多平台营销内容的企业（比如品牌Pop-up
，线下活动领取小样，品牌展会）

本地商户：帮助本地商家在公域做内容种草，提升商家曝光和为商家引流
